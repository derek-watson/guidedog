Guidedog requires a server to work, you cannot view HTML files directly in your file system. This means you want the URL to read something like http://mysite.com or //http://localhost:3000/ and NOT file:///Users/local/path/to/guidedog/example.html. Luckily, Guidedog comes with a simple Node server to get you started. To start the server, simply navigate to the root Guidedog folder in Terminal, and run `gulp`.

Note - You'll have to install Node in order to start the server.
